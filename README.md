## 研究方向
1. **均线策略 (Moving Average)** - SMA, EMA 交叉策略
2. **动量策略 (Momentum)** - RSI, MACD, 动量反转
3. **套利策略 (Arbitrage)** - 跨市场、跨品种
4. **宏观策略 (Macro)** - 经济周期、利率、通胀
