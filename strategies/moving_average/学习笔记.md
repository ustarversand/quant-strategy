# 均线策略学习笔记

## 什么是均线？

**均线 (Moving Average)** 是过去N天价格的平均值。

### 常见类型
| 类型 | 英文 | 特点 |
|------|------|------|
| 简单均线 | SMA | 简单平均，等权重 |
| 指数均线 | EMA | 近期权重更大，反应更快 |
| 加权均线 | WMA | 自定义权重 |

## 经典策略：均线交叉 (MA Crossover)

### 策略逻辑
- **金叉**：短期均线向上突破长期均线 → 买入
- **死叉**：短期均线向下突破长期均线 → 卖出

### 常用参数组合
| 组合 | 用途 |
|------|------|
| 9日 / 21日 | 短线交易 |
| 50日 / 200日 | 中长线趋势 |
| 10日 / 30日 | 中期趋势 |

## MACD (移动平均收敛/发散)

MACD 是均线的进阶版本：

- **DIF线** = 12日EMA - 26日EMA
- **DEA线** = DIF的9日EMA
- **MACD柱** = (DIF - DEA) × 2

### 信号
- DIF 上穿 DEA → 买入
- DIF 下穿 DEA → 卖出
