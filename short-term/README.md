# 短线交易系统 (练习盘感用)

## 理念
- 持有周期: 1天 - 1周
- 目标: 练习盘感，积累经验
- 原则: 轻仓练习，严格止损

## 交易规则

### 选股条件
1. 成交量放大 (>2倍均量)
2. 涨幅 > 5% 或跌幅 > -3%
3. 板块/题材热点

### 买入规则
- 首次仓位: 不超过 10%
- 买入后上涨 > 5% → 可加仓
- 买入后下跌 > -3% → 立即止损

### 卖出规则
- 盈利 > 8% → 止盈一半
- 盈利 > 15% → 全部止盈
- 持有超过 3天 → 强制平仓
